Проект скачивает Html-страницу, адрес которой ввели в косноль
 и выдает статистику уникальных слов.

Для работы нужно скачать и установить Intellij idea, с ver. java 1.8.

Далее открыть проект в IDE, запустить и следовать инструкции в консоле т.е
ввести адрес web-страницы в формате: "https://...", и в консоль программа 
выведет количество уникальных слов.
